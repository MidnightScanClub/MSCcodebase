Code for analysis of the MSC dataset

This codebase includes T1 segmentation, surface-based registration, myelin map creation, BOLD preprocessing, BOLD fc-processing, cifti creation, infomap-based network detection, gradient-based parcellation, and data reliability testing.

This code is provided "as is", without any warranty. We cannot provide technical support for this code. For general questions, please contact Nico Dosenbach (dosenbachn@wustl.edu).


Additional required packages not included here:

Infomap: www.mapequation.org
Brain connectivity toolbox: http://www.brain-connectivity-toolbox.net
